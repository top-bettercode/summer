Summer Framework is a simple framework that will help you to rapid development of java projects with
Spring Boot.

## Features

* 统一日志框架.

* 集成spring data jpa and mybatis.

* gradle开发插件.

* 通过测试自动生成接口文档.

* 打包工具.

* 代码生成工具.
